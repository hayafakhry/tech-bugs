Firewall Configuration with pfSense

Latest post • Network Security

Dalam post ini, saya akan berkongsi pengalaman pertama saya configure firewall menggunakan pfSense. pfSense adalah firewall open source yang powerful untuk network security.

Langkah-langkah Configuration:

1. Installation: Download pfSense ISO dan install pada dedicated machine atau virtual environment.

2. Initial Setup: Access web interface melalui default IP (192.168.1.1) dan setup admin password.

3. Interface Configuration: Define WAN, LAN, dan OPT interfaces mengikut network topology.

4. Firewall Rules: Create rules untuk allow/block traffic berdasarkan source, destination, port, dan protocol.

5. NAT Configuration: Setup port forwarding untuk services yang perlu accessible dari internet.

6. VPN Setup: Configure OpenVPN atau IPSec untuk remote access yang secure.

Best Practices:

• Always deny by default, allow by exception

• Regular backup configuration

• Enable logging untuk monitoring

• Keep system updated

Configuration firewall adalah skill penting dalam network security. Dengan pfSense, kita boleh build enterprise-grade security solution dengan zero cost.
