Firewall Configuration with pfSense

March 25, 2024 5 min read Lab, Security

Today I configured firewall rules using pfSense in a virtual lab environment. Set up NAT, port forwarding, and VPN access for a simulated network. Learned about rule ordering and logging. This hands-on lab helped me understand how firewall policies work in real-world scenarios and how to troubleshoot connectivity issues.

Next step: Implement the same setup using iptables on a Linux server and compare the two approaches.
